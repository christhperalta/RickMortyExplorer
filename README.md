

# RickMortyExplorer

RickMortyExplorer is a mobile application that allows users to explore the Rick and Morty universe by consuming its official API. The app lets users browse characters in an interactive and intuitive way.

It was built using Jetpack Compose for a modern UI, Navigation 3 for navigation, Retrofit for network requests, Hilt for dependency injection, and Room for local data persistence.
